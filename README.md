
## Setup and Run

1. Add `firebase-key.json` to resource

2. Build the Application

```bash
mvn clean install -DskipTests
```

3. Run with Docker

```bash
docker-compose up --build
```

4. Access the Application
- Backend API: http://localhost:8080 (or the port specified in .env)
